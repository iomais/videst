# Videoder

CLI + Interactive Media Downloader written in C++.

## Features
- Best Quality Download
- MP3 Extraction
- Playlist Support
- Custom Quality
- Progress Bar
- Multi-site support (via yt-dlp)

## Install (Debian/Ubuntu)

sudo apt install ./videoder_1.0_amd64.deb
Terminal: videoder URL
